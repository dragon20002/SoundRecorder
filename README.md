# SoundRecorder
Swift record sound &amp; get dB

Dev Env : Xcode 10.1, Swift 4.0 (iOS 9.3 cannot use recorder with Swift 4.2 current -Feb.27.19)<br>
Target Ver : >= iOS 9.3
